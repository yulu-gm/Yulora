# TASK-018 查找替换 intake

Task: TASK-018 查找替换

Goal: 提供最小但稳定的全文查找与替换能力，并保持 Markdown 文本、CodeMirror history 和 Electron 三层边界不被破坏。

In scope:
- 在 renderer 中建立查找 / 替换面板状态。
- 复用 CodeMirror search state 做全文匹配、高亮、上一项 / 下一项导航。
- 支持替换当前匹配与替换全部匹配。
- 覆盖替换进入 undo history 的回归测试。

Out of scope:
- 不新增正则、大小写开关、whole-word、跨文件搜索或搜索历史。
- 不新增 main / preload IPC 或任何 Node 能力。
- 不改变 Markdown parser、block map 或保存链路。

Landing area:
- `src/renderer/code-editor.ts`
- `src/renderer/code-editor-view.tsx`
- `src/renderer/editor/WorkspaceShell.tsx`
- `src/renderer/styles/app-ui.css`
- `src/renderer/styles/editor-source.css`
- 相关 Vitest 测试与项目记录文档

Acceptance:
- 支持当前文档全文查找，并在 CodeMirror 正文中高亮匹配。
- 支持上一项 / 下一项导航。
- 支持替换当前匹配和替换全部匹配。
- 替换操作可被 CodeMirror undo / redo 自然撤回。

Verification:
- `npm run test -- src/renderer/code-editor.test.ts`
- `npm run test -- src/renderer/editor/WorkspaceShell.test.tsx`
- `npm run typecheck`
- `npm run lint`
- `npm run test`
- `npm run build`
- `git diff --check`

Risks:
- CodeMirror search highlighter 依赖 search panel open state，本轮使用隐藏 panel 只启用高亮，不渲染 CodeMirror 默认搜索面板。
- 面板输入状态是 renderer ephemeral state；切换标签时必须清理 query，避免跨文档残留高亮。
