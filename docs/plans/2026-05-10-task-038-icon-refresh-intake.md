# TASK-038 图标清晰度与鱼身填充 Intake

## 背景

当前 Windows 桌面图标仍沿用 `assets/branding/*.svg` 作为唯一源文件，并通过 `scripts/generate-icons.mjs` 生成 `build/icons/light/icon.ico` 后进入 Windows 打包链路。

用户反馈两个问题：

- 桌面图标相比其他软件显得分辨率不够高。
- 图标中间鱼形区域是镂空效果，希望填充为黑色。

## 初步定位

- 当前 light 源 SVG 是白色圆形主体，鱼形区域主要通过透明镂空形成；生成后的 `icon-512.png` 在鱼身中心仍是白色/透明组合，不是黑色实体。
- 当前 `.ico` 只包含 `32/64/128/256` 四档，缺少 Windows 常见的 `16/24/48` 尺寸，桌面或快捷方式在部分缩放档位可能需要系统插值。

## 本轮范围

- 保持 SVG 仍是品牌资产唯一源。
- 调整 light/dark 源 SVG，使鱼身区域生成后为黑色不透明填充。
- 扩展 icon 生成尺寸，补齐 Windows 常用小尺寸。
- 补充生成器回归测试，验证 `.ico` 尺寸集合和 light 图标中心黑色填充。

## 不做

- 不更换整体 logo 造型。
- 不改打包发布版本号。
- 不执行正式 release。
