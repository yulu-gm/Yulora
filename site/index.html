设计哲学

少即是多，
文字为先

FishMark 不追求功能的堆叠，而是专注于让写作本身更稳定、更可靠。

Markdown 是事实来源
不自动重排文档，保存时忠实还原你的 Markdown 风格——缩进、换行、标点，一字不改。
WYSIWYM，而非 WYSIWYG
光标所在之处即回到 Markdown 源码。离开后渲染为阅读态。写作与阅读切换零摩擦。
本地优先，数据自主
文件存储在你的硬盘上，无需账号，无需云端同步，离线永远可用。你的文字，你来掌控。

块级渲染切换
标题、段落、列表、任务列表、引用块、代码块、分割线——非激活态渲染为视觉呈现，光标移入即回到源码。
多标签页工作区
单窗口内同时打开多个文档，支持标签排序、拖出成新窗口，每个标签独立管理保存状态。
新功能
可折叠大纲侧栏
基于标题层级自动生成大纲，右侧悬浮独立滚动，默认收起，点击可跳转至对应位置。
图片粘贴落盘
粘贴剪贴板图片自动写入本地 assets 目录，插入相对路径 Markdown，图片在编辑器中即时可见。
自动保存与外部变更检测
停止输入后自动保存，失焦时也会保存。外部程序修改文件时弹出提示，支持重载、保留内存版本或另存为。
主题与偏好设置
支持扫描 userData/themes 下的主题包，颜色模式、文档字体、字号、autosave 延迟均可在设置页实时调整。
行内格式渲染
粗体 **bold**、斜体 *italic*、行内代码 `code`、删除线 ~~strikethrough~~ 均在非激活行渲染为最终样式，光标所在行回到 Markdown 源码，兼容 IME 输入法。
